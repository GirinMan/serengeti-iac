Automated evaluation using LLM judges, G-Eval scoring, and custom judge model training (SFT + DPO). Grounded in the [LLM-as-a-Judge paradigm](https://arxiv.org/abs/2306.05685) (Zheng et al., NeurIPS 2023) with domain-specific bias mitigation. Evaluation pipeline adopts chain-of-thought scoring from [G-Eval](https://arxiv.org/abs/2303.16634) (Liu et al., EMNLP 2023).
