## 개요

AICA(AI Innovation Center Alliance) 정부 AI 데이터센터 지원 사업에 참여하여, **사업계획서 작성 → 인프라 확보 → 연구 과제 수행 → 분기별 성과 보고 → 우수성과자 선정**까지 전 과정을 주도적으로 수행했습니다.

## 사업 운영 전 과정

### 사업계획서 작성 & 신청
- 법률 AI 특화 GPU 컴퓨트 사업계획서 직접 작성
- 연구 목표·기대효과·일정·예산 설계
- 2024년 최초 신청 및 2026년 재신청 준비

### 인프라 확보 & 세팅
- H100×8 노드 자원 할당 정상화, 오프라인 서류 제출
- 방화벽/NAT 설정 직접 수행
- AICA vLLM 서빙 경로를 내부 aihub/LiteLLM 체인에 연결하는 아키텍처 설계

### 연구 과제 수행
- **2024**: 법률 특화 LLM 학습, 시맨틱 검색 모델, 문서 구조 분석 (H100×8, A100)
- **2025 H1**: 멀티모달 법률 Reasoning LLM Agent (H100×8)
- **2025 H2**: 법률 도메인 LLM-as-a-Judge 자동 평가 시스템 (A100×8)

### 성과 보고 & 관리
- 분기별 성과보고서 작성 및 제출
- 상용화 서비스 섹션 정리, 성과 지표 취합
- 팀 내 역할 분담 조율 및 최종 제출 관리

### 우수성과자 선정
- **2026 Q1**: 2025년 과제 **우수성과 사례집 선정** — 정부 평가에서 연구 성과 인정

## 주요 연구 성과

- GPU 지원사업 제안서 및 분기별 성과보고서 작성 주도
- 법률 특화 LLM 학습, 시맨틱 검색 모델, 문서 구조 분석 연구 과제 수행
- 2025년 과제 우수성과 사례집 선정 — 정부 평가에서 연구 성과 인정
- 정부 PoC 3건 수행 (공공기관 A, B, C)

## 기술 스택

H100 GPU, A100 GPU, PyTorch, Transformers, vLLM, LiteLLM, FAISS, FastAPI, Python
