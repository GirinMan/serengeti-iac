## Overview

Served 18 months as an Information Security Specialist (MOS 175104) in the Integrated Security Operations Center of the III Corps 103rd Signal Battalion Cyber Defense Unit, Republic of Korea Army. Managed intranet servers, UTM monitoring, and security log analysis in an air-gapped environment, and independently developed tools to automate repetitive tasks.

## Key Achievements

- Operated intranet servers and managed base infrastructure systems in an air-gapped environment
- Monitored Unified Threat Management (UTM) equipment and maintained security devices
- Performed security log collection and analysis
- Built automation tools from scratch using only standard libraries (no external packages)
- Torch Award for Excellence in Information Security, Class 5 (Jul 2020)
- Grand Prize in Army Core Values Contest, Poster Division (Sep 2020)
- Early promotion to Corporal (Dec 2020)
- Appointed Squad Leader (Apr 2021 – Aug 2021)
- Commendation from the Cyber Operations Command Commander (2025)

## Technical Approach

- Developed automation tools using only standard libraries, as the air-gapped environment prohibited installation of external libraries or packages
- Collected UTM equipment log data and detected anomalies through pattern analysis
- Improved operational efficiency by automating repetitive security inspections and reporting tasks
- Completed Information Security training curriculum
- Detailed technical procedures are restricted due to military classification

## Career Connection

Experience with integrated security operations, base infrastructure management, information security, procedure-based operations, and automation in constrained environments carried forward as a systems-level perspective in subsequent model serving, workflow, and production AI system operations.
