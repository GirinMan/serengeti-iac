## 개요

대한민국 육군 군단급 야전부대에서 정보보호병으로 18개월간 복무했습니다. 폐쇄망(air-gapped) 환경에서 내부망 서버 관리, UTM 관제, 보안 로그 분석을 수행하고, 반복 업무를 자동화하는 도구를 직접 개발했습니다.

## 주요 성과

- 폐쇄망 환경에서 내부망 서버 운영 및 기반체계 관리
- UTM(Unified Threat Management) 장비 관제 및 보안 장비 유지 관리
- 보안 로그 수집 및 분석 업무 수행
- 외부 패키지 없이 표준 라이브러리만으로 자동화 도구 직접 구현
- 정보보호 5기 성적우수 횃불상 수상 (2020.07)
- 육군 핵심가치 공모전 포스터 분야 최우수 (2020.09)
- 상등병 조기진급 (2020.12)
- 분대장 임명 (2021.04 – 2021.08)
- 사이버작전사령관 표창 (2025)

## 기술적 접근

- 폐쇄망 환경 특성상 외부 라이브러리나 패키지 설치가 불가능하여, 표준 라이브러리만으로 자동화 도구 개발
- UTM 장비의 로그 데이터를 수집하고 패턴 분석을 통한 이상 징후 탐지
- 반복적인 보안 점검 및 보고 업무의 자동화로 운영 효율 개선
- 정보보호 교육과정 이수
- 세부 기술 절차는 군사기밀에 해당하여 제한적으로 기술

## 커리어 연결점

통합관제, 기반체계 운영, 정보보호, 절차 기반 운영, 제한된 환경에서의 자동화 경험은 이후 모델 서빙·워크플로우·프로덕션 AI 시스템 운영에서의 시스템적 관점으로 이어졌습니다.
